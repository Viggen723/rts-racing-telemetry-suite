# Roy Telemetry Systems Racing Telemetry Suite
As my capstone project, I aim to bring an Android-based application paired with a homebrew ESP32 microcontroller-based telemetry logger to grassroots racers around the world. The goal is to create an open-source, wallet-friendly system that everyone can enjoy and hone their skills as a racer, be it car, or motorcycle racing. 

In this ever-changing AI world, the largest goal that I set for myself is to use **zero** AI, from the data structures and digital signal processing to this markdown file and the LaTeX documentation. There is too much available information from textbooks to online articles to miss the opportunity to learn a few cool things. :)
